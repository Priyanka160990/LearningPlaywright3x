# VS Code Shortcuts — Windows

## General

| Shortcut | Action |
|----------|--------|
| `Ctrl+Shift+P` / `F1` | Open Command Palette |
| `Ctrl+P` | Quick Open — search & open files |
| `Ctrl+Shift+N` | New VS Code window |
| `Ctrl+W` | Close current editor tab |
| `Ctrl+Shift+W` | Close current window |

## Editing

| Shortcut | Action |
|----------|--------|
| `Ctrl+X` | Cut line (no selection needed) |
| `Ctrl+C` | Copy line (no selection needed) |
| `Ctrl+V` | Paste |
| `Ctrl+Z` | Undo |
| `Ctrl+Y` | Redo |
| `Ctrl+Shift+K` | Delete entire line |
| `Ctrl+Enter` | Insert line below |
| `Ctrl+Shift+Enter` | Insert line above |
| `Alt+↑` / `Alt+↓` | Move line up / down |
| `Alt+Shift+↑` / `Alt+Shift+↓` | Copy line up / down |
| `Ctrl+D` | Select next occurrence of current word |
| `Ctrl+Shift+L` | Select all occurrences of current word |
| `Ctrl+F2` | Select all occurrences of current word (same as above) |
| `Ctrl+U` | Undo last cursor operation |
| `Shift+Alt+I` | Insert cursor at end of each selected line |
| `Ctrl+Shift+\` | Jump to matching bracket |
| `Ctrl+]` / `Ctrl+[` | Indent / Outdent line |

## Multi-Cursor & Selection

| Shortcut | Action |
|----------|--------|
| `Alt+Click` | Insert cursor at click position |
| `Ctrl+Alt+↑` / `Ctrl+Alt+↓` | Insert cursor above / below |
| `Ctrl+Shift+L` | Select all occurrences of current selection |
| `Shift+Alt+→` | Expand selection (smart) |
| `Shift+Alt+←` | Shrink selection (smart) |
| `Ctrl+A` | Select all |

## Navigation

| Shortcut | Action |
|----------|--------|
| `Ctrl+T` | Show all symbols (search across files) |
| `Ctrl+Shift+O` | Go to symbol in current file |
| `F12` | Go to definition |
| `Ctrl+Click` | Go to definition |
| `Alt+F12` | Peek definition (inline) |
| `Ctrl+Shift+F12` | Go to implementations |
| `Ctrl+Shift+O` | Go to symbol in file |
| `Ctrl+G` | Go to line number |
| `Ctrl+P` then `@` | Same as `Ctrl+Shift+O` (symbols) |
| `Ctrl+P` then `>` | Open Command Palette |
| `Ctrl+Tab` / `Ctrl+Shift+Tab` | Cycle through open editors |
| `Alt+←` / `Alt+→` | Navigate back / forward in history |

## Search & Replace

| Shortcut | Action |
|----------|--------|
| `Ctrl+F` | Find in current file |
| `Ctrl+H` | Find & replace in current file |
| `Ctrl+Shift+F` | Find in all files (sidebar) |
| `Ctrl+Shift+H` | Find & replace in all files |
| `F3` / `Shift+F3` | Find next / previous |
| `Alt+Enter` | Select all matches of find |
